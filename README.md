## 🚀 Featured Projects

### 🔶 [AWS DevOps Multi-Region Architecture](https://github.com/NitishK1/DevOps-Arch/tree/main/Project_AWS_DevOps)
A production-ready, multi-region AWS setup deploying a containerised Node.js
application with full CI/CD automation.

**Key highlights:**
- Multi-region deployment (us-east-1 & us-west-2) for high availability and DR
- Container orchestration with **ECS Fargate**
- Docker images managed via **Amazon ECR**
- Full CI/CD pipeline: **CodeCommit → CodeBuild → Staging (auto) → Approval
  gate → Production**
- Infrastructure as Code with **Terraform** (VPC, subnets, ALB, ECS, ECR,
  CodePipeline)
- Observability via **CloudWatch** dashboards, alarms, and **SNS** notifications

**Tech:** AWS ECS Fargate · ECR · CodeCommit · CodePipeline · CodeBuild ·
CloudWatch · SNS · Terraform

### 🔷 [Azure DevOps — Order Management System (ProjectX)](https://github.com/NitishK1/DevOps-Arch/tree/main/Project_Azure_DevOps)
End-to-end Azure DevOps implementation for a Node.js/Express.js Order
Management System with full project management and IaC.

**Key highlights:**
- Work tracking with **Azure Boards** (Epics, User Stories, Tasks)
- Source control in **Azure Repos**
- Multi-stage CI/CD pipelines with staging and production environments
- **Manual approval gates** before production deployments
- Infrastructure as Code with **Azure Bicep**
- Automated deployment scripts and rollback support

**Tech:** Azure DevOps · Node.js · Express.js · Azure Web Apps · Bicep · Azure
Boards · Azure Repos · Azure Pipelines
